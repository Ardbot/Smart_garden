# Проект

Умное земледелие.

Цель. Автоматизировать процессы на производственном участке. Информировать владельца/специалиста о состоянии хозяйства. 
Своевременно предупреждать о изменинии статуса.

## Технологии

Telegram bot на aiogram для информирования и управления
MQTT для обмена информацией между устройствами
Сервис Thingspeak для логирования значений

Интернет с WI-FI для связи ESP8266 устройств.
